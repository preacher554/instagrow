# Analytics Snapshots

This file stores human-readable analytics pulls for published InstaGrow content.

Machine-readable events should also be appended to `data/analytics-snapshots.jsonl` when automation exists.

## Snapshot Cadence

```text
T+1h   early velocity
T+24h  first-day signal
T+72h  stabilization
T+7d   weekly decision
T+30d  evergreen/long-tail review
```
