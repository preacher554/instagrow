# Account Metrics Log

This file stores account-level metrics pulled from Postiz, Outstand, or manual Instagram Insights extraction.

Machine-readable events should also be appended to `data/account-metrics-log.jsonl` when automation exists.

## Schema

```text
snapshot_time_wib:
client_account:
platform:
adapter:
social_account_id:
followers:
following:
media_count:
reach:
impressions:
views:
accounts_engaged:
total_interactions:
profile_visits:
clicks:
notes:
```
