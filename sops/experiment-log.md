# Experiment Log SOP

Use this format for every Instagram experiment.

## Experiment Template

```markdown
## Experiment: <short name>

Date:
Owner:
Account:
Platform:
Format: Reel / Carousel / Story / Static
Topic cluster:

### Hypothesis
If we publish <content type/angle>, then <audience segment> will <desired behavior> because <reason>.

### Content Details
- Hook:
- Core idea:
- CTA:
- Visual pattern:
- Caption keyword:

### Success Metrics
- Primary metric:
- Secondary metric:
- Minimum success threshold:

### Result
- Views:
- Completion rate:
- Saves:
- Shares:
- Comments:
- Profile visits:
- Follows/leads/replies:

### Decision
Scale / iterate / archive.

### Lesson
What should future agents remember?
```

## Decision Rules

- Scale if it beats baseline and aligns with monetization/trust.
- Iterate if signal is promising but hook/format needs work.
- Archive if low signal and weak strategic fit.
